# Dashboard Prototype - Skill Profile

ダッシュボードスタイルのプロトタイプ実装

## 📁 ファイル構成

```
dashboard/
├── common.css              # 共通スタイル（レイアウト、コンポーネント）
├── index.html              # ダッシュボード（ホーム）
├── profile.html            # プロフィール表示・編集
├── skills.html             # スキル一覧・管理
├── projects.html           # プロジェクト管理（未実装）
├── activities.html         # 活動ログ（未実装）
├── analytics.html          # 統計・分析（未実装）
├── growth.html             # 成長トラッキング（未実装）
├── settings.html           # 設定
└── README.md               # このファイル
```

## 🎨 デザインシステム

### カラーパレット

```css
--primary-500: #2196F3;      /* メインブルー */
--bg-primary: #0a0a0a;       /* ダークな背景 */
--bg-secondary: #1a1a1a;     /* カード背景 */
--text-primary: #ffffff;      /* メインテキスト */
--text-secondary: #999999;    /* セカンダリテキスト */
```

## 📱 レスポンシブ対応

### ブレークポイント

- **Desktop**: 1024px 以上
- **Tablet**: 768px - 1023px
- **Mobile**: 320px - 767px

### 挙動

- **Desktop**: サイドバー固定表示
- **Tablet**: サイドバー折りたたみ可能
- **Mobile**: サイドバーオーバーレイ表示

## 🛠️ 使い方

### ローカルで開く

1. ブラウザで `index.html` を開く
2. サイドバーのナビゲーションでページ移動

### サーバーで実行（推奨）

```bash
# Pythonの場合
python -m http.server 8000

# Node.jsの場合
npx serve

# ブラウザで開く
open http://localhost:8000/dashboard/
```

## 🎯 デザインの特徴

### 1. ダークモード
- 目に優しいダークカラースキーム
- 高コントラストで視認性が高い

### 2. モダンなカード UI
- ホバーエフェクト
- 適度なシャドウ
- 滑らかなトランジション

### 3. 直感的なナビゲーション
- アイコン+テキストで分かりやすい
- アクティブ状態が明確
- 折りたたみ可能

### 4. 情報の階層化
- 重要な情報を上部に配置
- カード型で情報をグループ化
- 視覚的な区別が明確

## 📝 カスタマイズ

### カラーテーマの変更

`common.css` の `:root` セクションで変数を変更：

```css
:root {
    --primary-500: #2196F3;  /* お好みの色に変更 */
    --bg-primary: #0a0a0a;   /* 背景色を変更 */
    /* ... */
}
```

### サイドバー幅の変更

```css
:root {
    --sidebar-width: 260px;  /* サイドバーの幅 */
}
```

## 🎨 Figmaとの連携

このプロトタイプは、Figmaデザインの実装版として作成されています。

### Figmaから実装へ

1. Figma Dev Modeでスタイル確認
2. `design-tokens.json` からカラー・スペーシング取得
3. コンポーネント構造をHTMLで再現
4. インタラクションをJavaScriptで実装

### デザイントークンの対応

| Figma | CSS変数 |
|-------|---------|
| Primary/500 | `--primary-500` |
| Neutral/900 | `--bg-primary` |
| Neutral/800 | `--bg-secondary` |
| Spacing/lg | `--spacing-lg` |
