**コンセプト**: 洗練・ミニマル・プロフェッショナル

#### 特徴
- **配色**: 白・黒・グレーのみ

- **雰囲気**: 落ち着いた、プロフェッショナル、洗練

#### 適用シーン

## テーマ切り替え

### モノトーン版

1. **Dark Mode（デフォルト）**
   - 背景: ダークグレー (#1a1a1a → #2d2d2d)
   - カード: 黒系グラデーション (#2a2a2a → #1f1f1f)
   - テキスト: 白・ライトグレー
   - 用途: メイン、夜間作業

2. **Light Mode**
   - 背景: ライトグレー (#f0f0f0 → #e0e0e0)
   - カード: 白系グラデーション (#ffffff → #f5f5f5)
   - テキスト: 黒・ダークグレー
   - 用途: 印刷、明るい環境

### カラフル版

1. **Primary** - 紫グラデーション（#667eea → #f093fb）
2. **Gold** - 金色（#f6d365 → #fda085）
3. **Silver** - 銀色（#a8edea → #fed6e3）
4. **Bronze** - ブロンズ（#ff9a56 → #ff6a88）
5. **Dark** - ダークモード（#141E30 → #243B55）

## 移行ガイド

### カラフル版からモノトーン版へ

変更点：
1. グラデーションをグレースケールに
2. アバターを円形→角丸四角に
3. スキルバーに数値パーセンテージ追加
4. ランクアイコンをシンプルに
5. 全体的なコントラストを抑える

### モノトーン版からカラフル版へ

変更点：
1. グレースケールをカラーグラデーションに
2. アバターを角丸四角→円形に
3. スキルバーの数値パーセンテージ削除
4. ランクアイコンを絵文字に
5. 全体的に彩度を上げる

## 最終推奨

### 主要ビュー: フルビュー

**プロジェクトのメインビューとして、包括的なフルビューを採用しました。**

理由：
- ✅ **包括性**: すべての要件を網羅
- ✅ **実用性**: 実際の使用シーンに対応
- ✅ **拡張性**: 新しいセクションを追加しやすい
- ✅ **プロフェッショナル**: ビジネス用途に最適
- ✅ **情報設計**: 見やすく整理されたレイアウト

### 補助ビュー: シンプルビュー

特定の用途向けの補助的なビューとして残しています。

用途：
- スキル評価レポート
- スキル可視化ツール
- プレゼンテーション資料
- 簡易プロフィール
